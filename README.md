## 🚀 How to run

### 📦 Install dependencies:
```console
gem install bundler jekyll
```
```console
bundle install
```

### 🛠 Run dev server:
```console
bundle exec jekyll s --livereload
```

## 📝 Create new post (ONLY FROM THE ROOT DIR)
